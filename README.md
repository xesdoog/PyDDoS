# About:

**PyDDoS** is a simple python script that launches Denial of Service (DoS) attacks on a specified URL.

_Does not include botnets._

# Requirements:

- [Python 3.12](https://www.python.org/downloads/)

# Usage:
######
    pyddos.py <url>
